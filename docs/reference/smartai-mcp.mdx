The Smart AI Documentation MCP Server allows you to search and retrieve information from the Smart AI documentation directly within your agent conversations.

## Set Up

### Configure Smart AI

1. Create a folder called `.smart-ai/mcpServers` at the top level of your workspace
2. Add a file called `continue-docs-mcp.yaml` to this folder
3. Write the following contents and save:

```yaml title=".smart-ai/mcpServers/continue-docs-mcp.yaml"
name: Smart AI Documentation MCP
version: 0.0.1
schema: v1
mcpServers:
  - uses: tommycrusher/smart-ai-docs-mcp
```

### Enable Agent Mode

MCP servers only work in agent mode. Make sure to switch to agent mode in Smart AI before testing.

## Troubleshooting

### MCP Server Not Loading

1. **Check configuration**: Ensure your YAML configuration uses the correct `uses` field with `tommycrusher/smart-ai-docs-mcp`
2. **Check agent mode**: MCP servers only work in agent mode
3. **Restart Smart AI**: Try restarting the Smart AI extension

### No Search Results

1. **Verify connection**: The MCP server needs internet access to search the documentation
2. **Check query format**: Try rephrasing your search query
3. **Test with known topics**: Search for well-documented features like "model configuration"
