## Steps to use a custom configuration in Smart AI

Once you've found the configuration you want to use on Smart AI Mission Control:

1. Click “Add Config” on its page
2. Add any required inputs (e.g. secrets like API keys)

After saving, open the Smart AI CLI or IDE extension and

- Select the configuration from the **config dropdown** in the Smart AI extension
- Type `/config` and select the configuration in the Smart AI CLI
- Begin using it for chat mode, agent mode, or other configured capabilities.
